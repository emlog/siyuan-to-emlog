## 命令

- pnpm install
- pnpm run build
- pnpm run dev

## 插件仓库

* [siyuan-to-emlog](https://github.com/emlog/siyuan-to-emlog)


## 发布流程

* 修改：package.json 版本号
* 修改： changelog.md
* 打标签：git tag v0.1.1
* 推送标签：git push origin v0.1.1
* 打包：pnpm run build
* 发布：github 上创建新的release
